# 論文作成：クイックスタートガイド

**目的**: 最短時間で`卒論_編集用.md`を`.卒論.tex`に反映する

---

## 🚀 5分で始める

### ステップ1: 表紙情報の反映（2分）

`.卒論.tex`の110-140行目を編集：

```latex
%論文題目
{\fontsize{20}{10.5}\selectfont
自然言語処理と解釈可能AIを活用した授業評価分析\\
～マルチタスク学習と順序回帰モデルによる満足度要因の特定～
}\\
```

`【指導教員名】`，`【役職】`，`【学籍番号】`，`【氏名】`を実際の値に置き換える

### ステップ2: 第1章の反映（3分）

`.卒論.tex`の156行目以降に追加：

```latex
\chapter{はじめに}\label{ch:introduction}
\section{研究の背景}
\subsection{教育の質向上の重要性}

近年，高等教育機関において，教育の質向上は重要な課題となっている．
学生の授業評価は教育改善の重要な指標であり，授業評価は多段階のスコアと自由記述で構成されている．
```

**Markdownからコピー＆ペースト**して，見出しだけLaTeXコマンドに変換

## 📋 変換表（コピー＆ペースト用）

### 見出しの変換

| Markdown | LaTeX |
|----------|-------|
| `## 第1章 はじめに` | `\chapter{はじめに}\label{ch:introduction}` |
| `## 第2章 関連研究` | `\chapter{関連研究}\label{ch:related_work}` |
| `## 第3章 研究方法` | `\chapter{研究方法}\label{ch:methodology}` |
| `## 第4章 マルチタスク学習による関係性の理解` | `\chapter{マルチタスク学習による関係性の理解}\label{ch:multitask}` |
| `## 第5章 SHAP分析による満足度要因の特定` | `\chapter{SHAP分析による満足度要因の特定}\label{ch:shap}` |
| `## 第6章 順序回帰モデルによる補完` | `\chapter{順序回帰モデルによる補完}\label{ch:ordinal}` |
| `## 第7章 おわりに` | `\chapter{おわりに}\label{ch:conclusion}` |
| `### 1.1 研究の背景` | `\section{研究の背景}` |
| `#### 1.1.1 教育の質向上の重要性` | `\subsection{教育の質向上の重要性}` |

### 強調の変換

| Markdown | LaTeX |
|----------|-------|
| `**強調**` | `\textbf{強調}` |

### 表の変換テンプレート

**Markdownの表**:
```markdown
| 統計量 | 値 |
|--------|-----|
| 平均 | 3.459 |
```

**LaTeXの表**:
```latex
\begin{table}[htbp]
\caption{表のタイトル}
\begin{center}
\begin{tabular}{ll}\Hline
統計量 & 値 \\ \hline
平均 & 3.459 \\ \Hline
\end{tabular}
\end{center}
\label{tab:label_name}
\end{table}
```

## ⚡ 効率的な作業方法

### 方法1: 章ごとに変換

1. 第1章だけを変換
2. コンパイルして確認
3. 第2章を追加
4. 以下，順次追加

### 方法2: 一括変換

1. すべての章を一度に変換
2. エラーを一括で修正
