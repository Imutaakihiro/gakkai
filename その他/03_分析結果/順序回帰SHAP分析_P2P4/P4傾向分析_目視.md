# P4（高評価確率）SHAP 傾向分析（目視）

**対象ファイル**: `word_importance_p4高評価確率_production.csv`  
**解釈**: P4 への SHAP 重要度が**正**＝その語が含まれると「4点寄り」になりやすい＝**高評価に寄与**する語とみなす。

**注意**: P4 の重要度の最大値は約 0.00054 であり、P2（約 0.003〜0.008）に比べて**約1桁小さい**。高評価に「特に効く」語は、本データ・モデルでは P2 ほどはっきりは分離されていない。

## 2. 傾向の整理（目視によるまとめ）

### （1）重要度の大きさと解釈の難しさ

- P4 の SHAP 重要度は **P2 の約 1/10〜1/15** 程度であり、「4点を増やす」ために強く効く語が少ない。
- 上位には **圏、無、句、唱、妙、ヵ月** など、単体では解釈しにくい語や断片的な語が多く、**授業・質問の文脈に依存**する。

### （2）「否定形で肯定」の可能性（高評価に寄与しうる語）

- **拒否、見下、偏見** など、通常はネガティブな語が P4 に寄与しているように見える。
- **「拒否感がない」「見下さない」「偏見がない」** のように、否定形で使われていると高評価の記述と共起しうる。生テキストでの共起確認があると解釈が明確になる。

### （3）授業内容・専門用語・固有名

- 出典、要請、拠点、佐竹、要人、エレクトロニクス、帝国、識別、政府、国民 など。
- 特定の授業・分野の自由記述に現れる語であり、**その授業で高評価が語られやすい文脈**と結びついている可能性がある。

### （4）ポジティブ・肯定文脈らしい語（中位以降に出現）

- **楽し、面白、楽しい、恵まれ、易い、役立っ、理想、進歩** などが、P4 に寄与する語のなかに含まれる。
- これらは「高評価に寄与する」と解釈すると自然であり、**高評価の自由記述で使われやすい語**と整合的である。
- ただし重要度は P2 の不満語に比べて小さく、**「4点を増やす」効果は P2 の「2点を減らす」効果ほどは強く表れていない**。

### （5）P2 と共通する要因（高圧、圏）

- **高圧** は P2・P4 の両方で重要度が高い。記述の文脈によっては低評価・高評価のいずれにも関与しうる**共通要因**と解釈できる。
- **圏** も両方で上位。単体の解釈は難しいが、何らかの文脈で両方の評価と結びついている可能性がある。

---

## 3. 結論（目視分析に基づく）

- **P4（高評価確率）に寄与する語**は、重要度が全体的に小さく、**「4点を増やす」ために特に効く語**は P2 の「2点を減らす」語ほどは明確には分離されなかった。
- 意味のある語のなかでは、**ポジティブ・肯定文脈らしい語**（楽し、面白、楽しい、恵まれ、易い、役立っ、理想、進歩）が P4 に寄与しており、高評価の自由記述と整合的である。
- **拒否、見下、偏見** などは、否定形での共起（「〜がない」「〜しない」）であれば高評価との共起と解釈しうる。
- P2・P4 の**強い共通要因**（高圧、圏）は、文脈によって低評価・高評価の両方に関与しうる語として扱うのが妥当である。
- 施策面では、**P2 の「低評価に寄与する語」を減らす**ことがまず優先され、P4 の語は「高評価の記述で使われやすい語」の手がかりとして補助的に参照するのが現実的である。

---

## 4. 注意（データの性質）

- トークン単位（WordPiece）のため、**断片**が多く、目視では「意味のある語」に絞って解釈した。
- P4 の重要度は P2 より約1桁小さいため、**ノイズの影響を受けやすく、解釈は P2 より慎重に**行う必要がある。
- 同一語の複数トークンは別行のため、語単位で集約すると順位が変わる可能性がある。
- 因果ではなく**相関**（その語が含まれる記述で P4 が高くなりやすい傾向）として読むのが妥当。
