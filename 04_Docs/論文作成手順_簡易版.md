# 論文作成手順：簡易版（手直しフェーズ用）

**目的**: `卒論_編集用.md`の内容を`.卒論.tex`に反映するための，実践的な手順

---

## 🎯 基本方針

1. **Markdownで内容を完成させる**（現在の状態）
2. **LaTeXに変換する**（手作業でコピー＆ペースト）
3. **LaTeXでコンパイルしてPDFを生成する**

---

## 📝 作業手順（章ごと）

### ステップ1: 表紙情報の反映

**場所**: `.卒論.tex`の110-140行目

**作業**:
1. `卒論_編集用.md`の9-29行目を確認
2. 以下の箇所を実際の値に置き換える：
   - `【指導教員名】` → 実際の氏名
   - `【役職】` → 実際の役職（教授，准教授など）
   - `【学籍番号】` → 実際の学籍番号（全角）
   - `【氏名】` → 実際の氏名

**例**:
```latex
指導　\fitwidth{7\zw}{山田太郎}　教授
\fitwidth{4.5\zw}{学籍番号}\hspace{1\zw}２２Ｍ１０９９\\
\fitwidth{4.5\zw}{氏名}\hspace{1\zw}\fitwidth{7\zw}{田中花子}
```

### ステップ2: 各章の反映

#### 2.1 章の開始

**Markdown**:
```markdown
## 第1章 はじめに
```

**LaTeX**:
```latex
\chapter{はじめに}\label{ch:introduction}
\newpage  % 章を新しくする場合はページを改める
```

#### 2.2 節の開始

**Markdown**:
```markdown
### 1.1 研究の背景
```

**LaTeX**:
```latex
\section{研究の背景}
```

#### 2.3 項の開始

**Markdown**:
```markdown
#### 1.1.1 教育の質向上の重要性
```

**LaTeX**:
```latex
\subsection{教育の質向上の重要性}
```

#### 2.4 本文のコピー

**Markdownの本文をそのままコピー**（句読点は全角「，」「．」）

**例**:
```latex
近年，高等教育機関において，教育の質向上は重要な課題となっている．
学生の授業評価は教育改善の重要な指標であり，授業評価は多段階のスコアと自由記述で構成されている．
```

### ステップ3: 表の変換

**Markdown**:
```markdown
| 統計量 | 値 |
|--------|-----|
| 平均 | 3.459 |
| 標準偏差 | 0.216 |
```

**LaTeX**:
```latex
\begin{table}[htbp]
\caption{授業評価スコアの基本統計量（n=3,268授業）}
\begin{center}
\begin{tabular}{ll}\Hline
統計量 & 値 \\ \hline
平均 & 3.459 \\
標準偏差 & 0.216 \\ \Hline
\end{tabular}
\end{center}
\label{tab:course_score_stats}
\end{table}
```

**本文中で参照**:
```latex
授業評価スコアの基本統計量を表\ref{tab:course_score_stats}に示す．
```

---

### ステップ4: 数式の変換

**Markdownの数式はそのまま使用可能**（LaTeX形式で記述されているため）

**インライン数式**:
```latex
$E[y] = 1 \times P_1 + 2 \times P_2 + 3 \times P_3 + 4 \times P_4$
```

**独立した数式（数式番号付き）**:
```latex
\begin{equation}
E[y] = 1 \times P_1 + 2 \times P_2 + 3 \times P_3 + 4 \times P_4
\label{eq:expected_value}
\end{equation}
```

**本文中で参照**:
```latex
期待値は式\ref{eq:expected_value}で計算される．
```

---

### ステップ5: 強調（太字）の変換

**Markdown**:
```markdown
**データレベルの一致が重要**
```

**LaTeX**:
```latex
\textbf{データレベルの一致が重要}
```

## 📋 変換チェックリスト

### 見出しの変換
- [ ] `## 第X章 ...` → `\chapter{...}\label{ch:...}`
- [ ] `### X.X ...` → `\section{...}`
- [ ] `#### X.X.X ...` → `\subsection{...}`

### 本文の変換
- [ ] 句読点が全角「，」「．」になっている
- [ ] 文体が「である」調で統一されている
- [ ] 強調（太字）が`\textbf{...}`に変換されている

### 表の変換
- [ ] `\begin{table}[htbp]`で開始
- [ ] `\caption{...}`でキャプション（表の上）
- [ ] `\label{tab:...}`でラベルを設定
- [ ] 本文中で`表\ref{tab:...}`として参照

### 数式の変換
- [ ] インライン数式は`$...$`で囲まれている
- [ ] 独立した数式は`\begin{equation}...\end{equation}`
- [ ] `\label{eq:...}`でラベルを設定
- [ ] 本文中で`式\ref{eq:...}`として参照

### 参考文献の変換
- [ ] 本文中で`\cite{label}`を使用
- [ ] 参考文献リストに`\bibitem{label}`を追加

## ⚠️ よくある間違い

### 間違い1: 句読点の混在

**間違い**:
```latex
近年，高等教育機関において，教育の質向上は重要な課題となっている.
```

**正しい**:
```latex
近年，高等教育機関において，教育の質向上は重要な課題となっている．
```

### 間違い2: 表の参照がない

**間違い**:
```latex
授業評価スコアの基本統計量を以下に示す．
\begin{table}...
```

**正しい**:
```latex
授業評価スコアの基本統計量を表\ref{tab:course_score_stats}に示す．
\begin{table}...
\label{tab:course_score_stats}
```

### 間違い3: 数式の変数が定義されていない

**間違い**:
```latex
期待値は$E[y] = 1 \times P_1 + 2 \times P_2 + 3 \times P_3 + 4 \times P_4$で計算される．
```

**正しい**:
```latex
期待値は$E[y] = 1 \times P_1 + 2 \times P_2 + 3 \times P_3 + 4 \times P_4$で計算される．
ここで，$E[y]$は期待値，$P_i$は評価スコア$i$の確率である（$i \in \{1, 2, 3, 4\}$）．
```

## 💡 効率的な作業のコツ

### コツ1: 章ごとに変換してコンパイル

1. 第1章だけを変換
2. コンパイルしてエラーがないか確認
3. 第2章を追加
4. 以下，順次追加

**メリット**: エラーを早期に発見できる

### コツ2: 表と数式は後でまとめて変換

1. まず本文をすべて反映
2. 表をまとめて変換
3. 数式をまとめて変換

**メリット**: 作業が効率的

### コツ3: 図は最後に追加

1. まず本文，表，数式をすべて反映
2. 図を作成（Python，Excelなど）
3. 図をPDF形式で保存
4. LaTeXに追加

**メリット**: 図の作成に時間がかかるため，最後にまとめて作業
