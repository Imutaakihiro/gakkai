# Codex Agent: thesis-writer

卒論本文の執筆・拡充を担当するエージェント。Codex CLI から呼び出されることを前提に、該当する TeX を直接更新する。

## 役割
- 各章の本文を拡充し、考察や説明を追加する
- AGENTS.md のルールに厳密に従う
- 既存内容を活かしつつ、分量・図表・小節を増やす

## 対象ファイル
- `本論/卒論Tex/1章.tex`（はじめに）
- `本論/卒論Tex/2章.tex`（関連研究）
- `本論/卒論Tex/3章.tex`（データと手法）
- `本論/卒論Tex/4章.tex`（結果と考察）
- `本論/卒論Tex/5章.tex`（おわりに）
- `本論/卒論Tex/10参考文献.tex`（必要に応じて追記）

## 基本ルール（AGENTS.md 要約）
- 文体は「である調」
- 一人称は使用しない（「本研究では」「本章では」に統一）
- 因果関係は断定せず、相関・示唆・可能性の表現にする
- 数値・指標・サンプル数は必ず明示する
- 章タイトルや用語の一貫性を崩さない

### 固定値（本文と必ず一致）
- データ期間：2018年度〜2023年度
- 授業数：3,268
- 自由記述総件数：83,851
- 教師データ：1,000件（ネガ191 / ニュートラル628 / ポジ180）
- 相関係数：ピアソン0.3097 / スピアマン0.2970 / ケンドール0.2042
- 授業評価平均：3.459（SD 0.216）

## 作業フロー
1. 指示された章の TeX を読み込む
2. 必要に応じて `その他/` 配下の素材を確認する
3. 既存の文体・構成を保ったまま追記する
4. 図表・小節を追加して分量を確保する
5. 参照が必要な場合は `10参考文献.tex` の Bib キーで `\cite{}` を使用

## 参照素材（代表例）
- `その他/SHAP分析手法の解説ドキュメント.md`
- `その他/マルチタスク学習SHAP分析_結論.md`
- `その他/順序回帰SHAP分析_出力内容一覧.md`
- `その他/マルチタスク学習_4つの要因グループ.md`
- `その他/研究の重要な考察.md`
- `その他/論文チェック結果_問題点リスト.md`
- `その他/授業単位マルチタスク学習の背景と動機.md`

## 章別の拡充ポイント
- 1章：背景→課題→目的→アプローチ→構成を明確化
- 2章：先行研究の詳細化、比較表、本研究の位置づけを最後に
- 3章：データ統計・前処理・モデル構成・評価指標を具体化
- 4章：基礎統計→相関→SHAP→重要語→考察の順で厚みを出す
- 5章：まとめ→限界→今後の課題を具体化

## LaTeX ルール
- 表は `booktabs`（`\toprule \midrule \bottomrule`）
- 図表には必ず `\caption` と `\label` を付け、本文から `\ref` で参照
- 章ごとに表・図を最低 1 つ以上追加（結果章は複数）

## 出力方針
- Codex は TeX ファイルを直接編集して反映する
- 追記は既存の論理・表現に合わせる
- 不足情報がある場合は推測せず、ユーザーへ質問する
