# 卒論執筆ルール（Codex用）

このファイルは、`本論/卒論Tex` 配下の卒論本文を作成・修正する際の統一ルールを定める。

## 1. 分量目標（目安）
- はじめに：3ページ
- データの説明：4〜5ページ
- 手法の説明：7〜8ページ以上（内容量に応じて増加可）
- 結果：10ページ以上
- おわりに：2ページ
- 参考文献：2ページ以上（分量に応じて増加可）
- 付録：必要に応じて追加

※「ページ数」はおおよその目安。本文量が不足する場合は、表・図・小節の追加で補う。

## 2. 文体・表現
- 文体は「である調」。
- 一人称（私、我々）は使用しない。主語は「本研究では」「本章では」等に統一。
- 因果関係は断定せず、相関・示唆・可能性の表現を用いる。
- 数値・指標・サンプル数は明示する。
- 研究対象、期間、データ規模は必ず具体的に記述する。

## 3. 章構成の基本方針
### 第1章 はじめに
- 背景 → 課題 → 研究目的 → 本研究のアプローチ → 論文構成
- 1章だけで研究の全体像がわかるようにする。

### 第2章 関連研究
- 授業評価研究 / 感情分析 / BERT / マルチタスク学習 / SHAP / 順序回帰
- 「本研究の位置づけ」を必ず最後に置く。

### 第3章 データと手法
- データセット概要、アンケート構成、前処理、教師データ、モデル構成、評価指標
- 図や表でデータの規模・分布・ラベル比率を示す。

### 第4章 結果と考察
- 基礎統計 → 相関分析 → SHAP結果 → 重要語・要因タイプ → 考察
- 結果は表・図を積極的に入れて分量を確保する。

### 第5章 おわりに
- まとめ → 限界 → 今後の課題

## 4. 分量確保のための必須要素
- 各章に **複数の小節** を設ける。
- 章ごとに **表・図を最低1つ** 以上入れる（図が無い場合は表で代替）。
- 結果章は **表・図を複数** 配置し、語彙リストや指標比較を記載する。
- 手法章は **モデル図・フロー図・擬似コード** の挿入を検討する。

## 5. 参照・引用
- 文献は `10参考文献.tex` の Bib キーを使用して `\cite{}` で統一。
- 新規文献を追加する場合は `10参考文献.tex` に必ず追記する。

## 6. LaTeXルール
- 各章ファイルは `本論/卒論Tex/` 配下の `1章.tex`〜`5章.tex` を編集する。
- 表は `booktabs` を使用（`\toprule \midrule \bottomrule`）。
- 図表には必ず `\caption` と `\label` を付け、本文から `\ref` で参照する。

## 7. 既存データとの整合性
- 以下の数値は本文と一致させる（必要に応じて修正）：
  - データ期間：2018年度〜2023年度
  - 授業数：3,268
  - 自由記述総件数：83,851
  - 教師データ：1,000件（ネガ191 / ニュートラル628 / ポジ180）
  - 相関係数：ピアソン0.3097 / スピアマン0.2970 / ケンドール0.2042
  - 授業評価平均：3.459（SD 0.216）

## 8. 禁止事項
- 因果関係の断定（「〜の原因である」など）。
- 主観的表現（「〜だと思う」「〜と感じた」）。
- 章タイトルや用語の一貫性を崩す改変。
